# lerna、yarn workspace 最佳实践

## Description

#### 【工程化建设】基于 Lerna 包管理、Yarn 依赖管理 - Monorepo 项目的最佳实践

- lerna 项目构建
- yarn workspace 依赖管理
- commitizen && cz-lerna-changelog 优雅的代码提交
- commitlint && yorkie 提交规范
- standardjs && lint-staged 代码规范
- 日志自动生成
- 编译、压缩、调试

## Project setup

```
yarn install
```

## Clean node_modules

```
yarn  clean
```

## Commit

```
yarn commit
```

## Publish

```
yarn pub
```

## Build

```
yarn build
```

## Test

```
yarn test
```
